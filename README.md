🩺 Breast Cancer Prediction System using Artificial Neural Network (ANN)

📌 Overview
This project is a Deep Learning-based Breast Cancer Prediction System developed using the Breast Cancer Wisconsin Diagnostic Dataset. The model predicts whether a breast tumor is Benign or Malignant based on 30 diagnostic features using an Artificial Neural Network (ANN) built with TensorFlow/Keras.

🎯 Features
The prediction is based on the following diagnostic features:
Radius
Texture
Perimeter
Area
Smoothness
Compactness
Concavity
Concave Points
Symmetry
Fractal Dimension
And 20 additional diagnostic features (30 features in total)

🔄 Project Workflow
Imported the required Python libraries (NumPy, Pandas, Matplotlib, Scikit-learn, TensorFlow, and Keras)
Loaded and explored the Breast Cancer Wisconsin Diagnostic Dataset
Performed Exploratory Data Analysis (EDA) and statistical analysis
Checked for missing values and analyzed the target variable distribution
Split the dataset into training and testing sets
Standardized the feature values using StandardScaler
Built and trained an Artificial Neural Network (ANN)
Evaluated the model using training and testing accuracy
Visualized training and validation accuracy and loss
Built a predictive system that classifies tumors as Benign or Malignant
